== Description ==

ResQ is a collaborative plugin, which allows any website to offer its visitors the opportunity to help those in need or to benefit from local help. It will allow your users to get redirected to pre-existed, trusted, charities or self-help networks.
 
It was built as an initiative to help fight the effects of Covid-19 virus and thought as a way for websites to engage in causes that match their values, and eventually develop their CSR.
 
Major features in ResQ allow the user to be redirected to pages that enable him/her to:
1. ask for help.
2. help by volunteering time, donating blood, money or material.



== Installation ==

Download ResQ on your website, activate it.
1, 2, 3 : You’re ready!
(You can personalize it through the back-end of your website)
